# Lendsqr Frontend Engineer Assessment

This project is a frontend implementation of the Lendsqr Admin Console
assessment using React, TypeScript, and SCSS.

## Live Demo
https://liadi-ayomide-daniel-lendsqr-fe-test.vercel.app

## GitHub Repository
https://github.com/AYOMIDE-LIADI/lendsqr-fe-test

## Tech Stack
- React
- TypeScript
- SCSS
- Vite
- Mock API (Mockaroo / Mocky)
- LocalStorage

## Features
- Login page with email validation
- Dashboard with user statistics
- Users page with:
  - 500 mock users
  - Filtering
  - Pagination
  - Action menu
  - light and dark mode
- User details page with:
  - Persistent data via localStorage
  - Tab navigation
- Fully responsive (mobile, tablet, desktop)

## Project Structure
- `components/` – reusable UI components
- `pages/` – page-level components
- `api/` – mock API logic
- `styles/` – SCSS variables, mixins, and globals
